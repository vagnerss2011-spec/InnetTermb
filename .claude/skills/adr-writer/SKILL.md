# ADR Writer Skill

Use esta skill quando uma decisão técnica afetar arquitetura, segurança, contratos, banco de dados, protocolo ou distribuição.

## Passos

1. Identifique contexto e problema.
2. Liste alternativas consideradas.
3. Escreva decisão.
4. Descreva consequências positivas e negativas.
5. Adicione critérios de reversão.
6. Salve em `adr/ADR-XXX-titulo.md`.

## Template

```markdown
# ADR-XXX — Título

## Status

Proposta | Aceita | Substituída

## Contexto

## Decisão

## Consequências positivas

## Consequências negativas

## Alternativas consideradas

## Critérios de revisão
```
